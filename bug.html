This HTML snippet demonstrates an uncommon error: using a reserved keyword as an attribute name.

<div class="my-div" for="my-element">This is some text</div>

The attribute for is a reserved keyword in HTML (it's used to associate a label with a form element). While browsers might not always throw an error, using it incorrectly could lead to unexpected behavior and hinder JavaScript interactions with the element. It's best to avoid using reserved words as attributes and opt for descriptive custom names.
